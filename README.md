# Portfolio Website - Arohi Jadhav

A world-class, interactive portfolio website built with React, TypeScript, Tailwind CSS, and GSAP.

## Tech Stack

- **React** with **Vite**
- **TypeScript**
- **Tailwind CSS**
- **GSAP** with ScrollTrigger plugin
- **Shadcn/ui** style components (built manually with Radix UI primitives)

## Visual Identity

- **Background**: Pure Black
- **Cards/UI**: Dark Gray (zinc-900)
- **Primary Text**: White
- **Secondary Text**: Light Gray (zinc-400)
- **Accent**: Electric Pink/Fuchsia
- **Typography**: Sora (headings), Inter (body)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
